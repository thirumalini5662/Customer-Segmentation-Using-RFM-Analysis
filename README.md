# Customer-Segmentation-Using-RFM-Analysis

A Customer segmentation project using the RFM (Recency, Frequency, Monetary) model to identify Loyal, Active, At-Risk, and Lost customers. Built an interactive Power BI dashboard with KPI tracking, customer behavior analysis, revenue contribution insights, and data-driven marketing recommendations.

Project Overview

This project focuses on Customer Segmentation using the RFM (Recency, Frequency, Monetary) model. The objective is to classify customers based on purchasing behavior and help businesses design targeted marketing strategies.

Business Problem

Businesses often struggle to identify loyal customers, retain at-risk customers, and optimize marketing campaigns. RFM analysis provides a structured approach to segment customers according to engagement and spending patterns.

Key Metrics

- Recency (Days since last purchase)
- Frequency (Number of purchases)
- Monetary Value (Total customer spending)
- RFM Score
- Customer Segment

Customer Segments

- Loyal Customers
- Active Customers
- At Risk Customers
- Lost Customers

Dashboard Features

- Total Customers KPI
- Total Revenue KPI
- Average Monetary Value
- Average Recency
- Average Frequency
- Customer Segment Distribution
- Revenue Contribution by Segment
- RFM Score Heatmap
- Customer Value Analysis
- Revenue Share Treemap

Tools Used

- Power BI
- Microsoft Excel
- DAX
- Data Modeling

Business Insights

- Identified high-value loyal customers for retention programs.
- Detected at-risk customers requiring re-engagement campaigns.
- Analyzed revenue contribution across customer segments.
- Provided actionable recommendations to improve customer lifetime value.

Outcome

The project demonstrates customer analytics, KPI design, business intelligence reporting, dashboard development, and data-driven decision-making skills relevant to Business Analyst and Data Analyst roles.
